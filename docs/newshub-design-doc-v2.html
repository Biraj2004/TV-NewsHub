NewsHub — full design doc

Companion to Architecture.md / Build.md / Design.md. Dark, remote-first UI for Android TV / Google TV, spec'd for both FHD and 4K panels. Country selects the channel universe, language filters within it. Every tile carries a live/offline/checking status. Every non-happy-path state below has an explicit screen — nothing in this app should ever render blank.

3. Edge cases

Nothing renders blank. Each of these is a real, designed screen — see Design-full.md §4 for the full table.

Full-screen state, not a toast. Silent background retry — dismisses automatically the moment connectivity returns.
