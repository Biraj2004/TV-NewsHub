NewsHub — TV app design doc

Dark, remote-first UI for Android TV / Google TV. Country selects the channel universe, language filters within it. Focus state = thick white border + slight scale. All overlays fade in on input and fade out after a few seconds idle.

1. Home screen

Country row (new) sits above language, so the whole channel set changes with it. India is default/focused on first launch.

3. Player — controls visible

Shown on open, or for ~4-5s after any remote press. Play/pause, previous/next channel, back to grid. Fades in ~200ms.

4. Player — controls hidden (idle)

Auto-hides after the idle timer elapses with no remote input. Fades out ~300ms, leaving only video. Any key press brings it back.
